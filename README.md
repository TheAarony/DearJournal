# DearJournal

 Welcome to DearJournal, 
  
A private web application designed to allow users to write down their thoughts, ideas and goals

This web application was built using **ASP.NET MVC** framework and **C#** for the backend. I have used **HTML**,**CSS** and **Razor** to build a simple user interface for the frontend. Used ** Entity Framework code migration** to build the database architecture, which can help with scalability.
